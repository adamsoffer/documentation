<Note>
  This page is a placeholder tutorial. It will walk through getting an API key, installing the Livepeer SDK (Node or Python) or using curl, and making a first call.
</Note>

<Steps>
  <Step title="Create an API key">
    Sign in to the Livepeer dashboard and create a new API key.
  </Step>
  <Step title="Install the SDK">
    Install the Node, Python, or other SDK — or skip this step and use curl directly.
  </Step>
  <Step title="Run your first call">
    Send a request to a Livepeer pipeline and inspect the returned run.
  </Step>
  <Step title="Inspect the run in the dashboard">
    Open the run in the dashboard to see latency, cost, and where it ran.
  </Step>
</Steps>
