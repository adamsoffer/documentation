<Note>
  This page is a placeholder tutorial. It will walk through installing the Livepeer MCP server in Cursor, signing in, and running a first capability call from the editor.
</Note>

<Steps>
  <Step title="Install the Livepeer MCP server in Cursor">
    Add the server to Cursor's MCP settings using the documented config block.
  </Step>
  <Step title="Sign in and create an API key">
    Open the dashboard, create a key, and paste it into the MCP server config.
  </Step>
  <Step title="Run your first call">
    Ask Cursor to call a Livepeer pipeline and inspect the returned run.
  </Step>
  <Step title="Inspect the run in the dashboard">
    Open the run in the Livepeer dashboard to see latency, cost, and where it ran.
  </Step>
</Steps>
