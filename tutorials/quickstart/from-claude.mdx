<Note>
  This page is a placeholder tutorial. It will walk through installing the Livepeer MCP server in Claude Code or Claude Desktop, signing in, and running a first capability call.
</Note>

<Steps>
  <Step title="Install the Livepeer MCP server in Claude">
    Add the server to your Claude Code or Claude Desktop MCP configuration.
  </Step>
  <Step title="Sign in and create an API key">
    Open the dashboard, create a key, and paste it into the MCP server config.
  </Step>
  <Step title="Run your first call">
    Ask Claude to call a Livepeer pipeline and inspect the returned run.
  </Step>
  <Step title="Inspect the run in the dashboard">
    Open the run in the Livepeer dashboard to see latency, cost, and the orchestrator that handled it.
  </Step>
</Steps>
