Real-time AI video on the open Livepeer Network. Run remote inference in a few lines — or publish your own pipeline.

<Tabs>
  <Tab title="Python">
    ```python
    from livepeer import Livepeer

    client = Livepeer(api_key="lp_...")
    stream = client.streams.start(
        model="daydream/video",
        prompt="cinematic",
    )
    print(stream.playback_url)
    ```
  </Tab>
  <Tab title="Node">
    ```javascript
    import { Livepeer } from "@livepeer/sdk";

    const client = new Livepeer({ apiKey: "lp_..." });
    const stream = await client.streams.start({
      model: "daydream/video",
      prompt: "cinematic",
    });
    console.log(stream.playbackUrl);
    ```
  </Tab>
  <Tab title="cURL">
    ```bash
    curl -X POST https://api.livepeer.org/v1/stream/start \
      -H "Authorization: Bearer $LIVEPEER_API_KEY" \
      -H "Content-Type: application/json" \
      -d '{"model":"daydream/video","prompt":"cinematic"}'
    ```
  </Tab>
</Tabs>

<Note>
  Code samples mirror the dashboard's example API shape (POST `/v1/stream/start` with `model` + `prompt`). Final SDK reference will land in the content pass.
</Note>
